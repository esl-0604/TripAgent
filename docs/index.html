STEP 1

Slack 초대 요청

회사 이메일을 제출해주세요. 수합 후 일괄로 Endorobotics Slack 워크스페이스에 초대됩니다.

제출 시 메일 앱이 열립니다. 열린 메일에서 "보내기" 버튼을 눌러 전송해주세요.
문의: eunsang.lee@endorobo.com

📬 초대 메일을 받으면

  1. 메일 본문의 "Join Now" 버튼 클릭 → Slack 가입
  2. Slack 앱 모바일·데스크탑 설치 (권장)
  3. #출장일지 채널로 입장 → 아래 출장 전 준비 따라하기
STEP 2

출장 전 준비 (5분)

떠나기 전 딱 세 가지만 확인하면 됩니다.

  1. 1

    #출장일지 채널 입장

    Slack 좌측 사이드바 → 채널 섹션에서 #출장일지 찾기. 없으면 상단 검색창에 입력 후 Join.

  2. 2

    본인 출장 쓰레드 찾기

    각 출장마다 parent 메시지가 미리 올라와 있습니다. 형식:

    YYMMDD-DD, {나라}, {행사명}

    예시: 260501-07, 미국, DDW 2026

  3. 3

    Dropbox 모바일 앱 설치

    Dropbox 앱을 폰에 설치하고 회사 계정으로 로그인. !대용량 기능을 쓰려면 필수입니다.

STEP 3

출장 중 사용법

명령어는 반드시 출장 쓰레드 안에서만 동작합니다. 채널 root에서 쓰면 무시됩니다.

💬

기본 사용: 쓰레드에 기록 던지기

출장 쓰레드에 텍스트·사진·짧은 영상을 자유롭게. 저녁 피드백 미팅에서 이게 그대로 자료가 됩니다.

  • • 관찰한 제품·경쟁사·의사 코멘트
  • • 슬라이드·부스·세션 사진
  • • 짧은 영상 (Slack 기본 업로드 한계 1GB 이하)
📤

!대용량 — 대용량 파일 업로드

1GB 넘는 영상(수술 녹화·강의 풀영상 등)에 사용.

1출장 쓰레드 안에서 !대용량 입력 후 전송
2봇이 "드랍박스 앱 열기" 링크를 답장으로 줌
3폰에서 그 링크 탭 → Dropbox 앱이 자동으로 오늘 Day N 폴더 열림
4폴더 안 + 버튼 → 파일 선택 → 업로드 시작
5폰 잠그거나 앱 전환해도 백그라운드에서 업로드 계속

업로드 완료 시 봇이 쓰레드에
"{본인} 님이 {Day N}에 파일명.mp4 업로드 완료" 알림을 돌려줍니다.

🗂️

!아카이브 — 하루 마무리

하루 일정 끝에 쓰레드 내용을 Dropbox로 동기화. 저녁 피드백 미팅 전 실행 권장.

• 쓰레드 전체 댓글이 messages.md로 정리되어 Day N 폴더에 저장

• 첨부 파일(사진·문서 등)도 동일 폴더로 업로드

• 여러 번 실행해도 중복 업로드 없음 (안전)

🌙 저녁 피드백 미팅

!아카이브 이후 Day N/messages.md 파일이 오늘의 기록을 구조화해줍니다. 팀 전원이 폴더 열고 미팅 시작 — 각자 올린 내용이 시간순으로 정리되어 있어 어젠다가 자동으로 잡힙니다.

잘 쓰는 팁

이 팁만 지키면 출장 데이터 품질이 2배.

✓ 현장에서 바로 올리기

"나중에 정리해야지"는 실패 공식. 보고 들으면 10초 안에 스레드에 던지기.

✓ 사진엔 맥락 한 줄

사진만 올리지 말고 "누가 발표하는 슬라이드" 같은 설명 함께. 저녁에 "이거 뭐였더라" 방지.

✓ 출처 명시

"Saito 선생님이 말하길..." 식으로 누가 말한 내용인지 본문에 명시. 보고서 작성 시 인용이 쉬움.

✓ 본인 관점 추가

관찰만 말고 "우리쪽 시사점은..." 한 줄 덧붙이기. 팀원들이 당신의 해석을 볼 수 있음.

✗ 개인 DM에 올리지 말기

반드시 출장 쓰레드에. 개인 메모는 팀이 못 봅니다.

✗ 명령어 단독으로만

!대용량만 단독 전송. "!대용량 부탁해요" ❌ / "!대용량" ⭕

FAQ

초대 메일 받는 데 얼마나 걸리나요?

이메일 제출 후 보통 1영업일 내 발송됩니다. 이틀 넘도록 안 오면 스팸함 확인 후 이은상에게 DM 주세요.

출장 쓰레드는 누가 만드나요?

각 출장은 사전에 관리자가 parent 메시지를 올려둡니다. 팀원은 해당 쓰레드에 댓글로 참여만 하면 됩니다.

Dropbox 폴더 구조가 어떻게 되나요?

전략기획 / 400 시장분석 / 000 학회 자료 / {출장제목} / {Day N}
예: 260501-1st Day, 260502-2nd Day.

!대용량 링크가 모바일에서 앱으로 안 열려요.

Dropbox 모바일 앱이 설치되어 있고 회사 계정으로 로그인된 상태여야 합니다. 미설치 시 웹페이지로 열리므로 반드시 앱 설치해주세요.

과거 출장 자료는 어디서 보나요?

Dropbox의 000 학회 자료 폴더 하위에 모든 과거 출장이 연도·행사별로 남아있습니다. 팀 공유 폴더라 누구나 열람 가능.

출장 쓰레드에 올린 메시지를 수정·삭제할 수 있나요?

Slack 기본 기능으로 본인 메시지는 언제든 수정·삭제 가능. 단 !아카이브 이미 실행된 내용은 Dropbox에도 반영되어 있으니, 수정 후 !아카이브 다시 실행하면 Dropbox도 갱신됩니다.
